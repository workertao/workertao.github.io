最近很懒，不想多说话

相机功能

调用：
Intent intent = new Intent(AcodePhotoShowActivity.this, AcodeCameraActivity.class);
startActivityForResult(intent, AcodeCameraConfig.TAKE_PHOTO_REQUEST);

返回数据：
@Override
protected void onActivityResult(int requestCode, int resultCode, Intent data) {
    super.onActivityResult(requestCode, resultCode, data);
    if (data == null) {
        return;
    }
    switch (requestCode) {
        case AcodeCameraConfig.TAKE_PHOTO_REQUEST:
            //获取绑定的数据源
            Bundle bundle1 = data.getBundleExtra("bundle");
            //获取绑定的返回的数据源
            String photoPath = bundle1.getString("photo_path");
            showImage(photoPath);
            break;
    }
}

相册功能（大图浏览，选中数量）

调用：
Intent intent = new Intent(AcodePhotoShowActivity.this, AcodePhotoListActivity.class);
startActivityForResult(intent, AcodeCameraConfig.SELECT_VP_REQUEST);

返回数据：
@Override
protected void onActivityResult(int requestCode, int resultCode, Intent data) {
    super.onActivityResult(requestCode, resultCode, data);
    if (data == null) {
         return;
    }
    switch (requestCode) {
        case AcodeCameraConfig.SELECT_VP_REQUEST:
             //获取绑定的数据源
             Bundle bundle = data.getBundleExtra("bundle");
             //获取绑定的返回的数据源
             ArrayList<ImagePhoto> imagePhotos = (ArrayList<ImagePhoto>) bundle.getSerializable("selectPhotoData");
             for (int i = 0; i < imagePhotos.size(); i++) {
             ImageView imageView = new ImageView(this);
             Glide.with(this)
                        .load(imagePhotos.get(i).getPath())
                        .into(imageView);
                linearLayout.addView(imageView);
             }
        break;
    }
}

Banner轮播

XML调用(自定义view)：

<com.acode.img.lib.viewpager.weigt.banner.AcodeBannerVp
     android:id="@+id/acode_banner_vp"
     android:layout_width="match_parent"
     android:layout_height="200dp" />

代码调用：
AcodeBannerVp acodeBannerVp = (AcodeBannerVp) findViewById(R.id.acode_banner_vp);
//设置banner图片数据源
acodeBannerVp.setData(urls)
         //设置banner文本内容数据源
        .setAcodeTitleText(urls1)
         //设置下标点的位置
        .setAcodePointGravity(AcodeVpConfig.RIGHT)
        //设置首次展示的banner图下标
        .setFirstItem(47)
        //设置是否自动轮播
        .setIsAutoPlay(true)
        //设置图片切换动画，已经封装，直接输入数字1-15即可
        .setAcodeVpAnim(5)
        //开始执行
        .start();
