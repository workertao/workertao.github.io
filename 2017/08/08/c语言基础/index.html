指针

int a = 10;
int *p;
p = &a;

※代表一个指针，&用来获取内存地址。
int ※p是一个指针变量，p=&a指得是p指向a的内存地址,※p的含义是指向p的内存地址所指向的值。
这块※p和p的含义也是纠结了老半天，对于初学者来说，确实有点坑。下边代码做下记录。

int main(){
    int a = 10;
    int *p;
    p = &a;
    printf("第一次p的值：%#x \n", p);
    printf("第一次*p的值：%d \n", *p);
    *p = *p + 10;
    printf("第二次p的值：%#x \n", p);
    printf("第二次*p的值：%d \n", *p);
    printf("a的值：%d \n", a);
}

打印结果：

%#x是打印内存地址的占位符，其实这点我感觉挺烦的，c语言内部做个匹配多好，反正变量是什么类型，系统是知道的，对应打出来不就行了，还要人为去操作。接着写，第一次p的值和第二次p的值打印的内存地址是一致的，没问题。第一次※p的值，和第二次※p的值发现由10变成了20，而第一次的10是哪来的呢？上边我们说了p取的a的内存地址，※p取的p的内存地址上的值。所以我们可以得出※p其实取的是a的值。最后我们再看打印的a的值：20。可我们不是定义的int a = 10吗。怎么打印变成20了，仔细想想，因为※p=20，※p指向的是p的内存地址上的值，而a的内存地址跟p是一样的，也就是※p其实指向的也是a的内存地址上的值。现在是不是一目了然了，完美！

*p++

int main(){
    int a = 10;
    int *p;
    p = &a;

    printf("*p的值： %#x \n", *p);
    *p++;
    printf("*p++后的值： %#x \n", *p);

    system("pause");
    return 0;
}

打印结果：

这里就要用到c语言的运算符的优先级了，++的优先级高于*，所以先执行p++。上边已经运行了p++，可以看到运行结果(上图c_04.jpg)，p++是内存地址的++。※p取的是内存地址上的值。因为没有这个内存地址，所以没有取到值。

(*p)++

int main(){
    int a = 10;
    int *p;
    p = &a;

    printf("*p的值： %d \n", *p);
    (*p)++;
    printf("*p++后的值： %d \n", *p);

    system("pause");
    return 0;
}

打印结果：

同样的，(※p)++中()运算符的优先级要高于++，所以先执行※p。取p的内存地址上的值，在执行++。

指针和数组

int main(){
    //数组名 就是数组的首地址
    int a[5] = { 905, 202, 302, 402, 502 };
    int *p;
    p = a;

    printf("p的地址：%#x \n", p);
    printf("*p的值：%d \n", *p);

     //a+5 ，表示a 移动到数组的第5个位置，即数组的长度
    int i;
    int len = sizeof(a)/sizeof(int);
    for (i = 0; i < len; i++){
        printf("a的地址：%#x \n", p);
        printf("a[i]的值：%d \n", a[i]);

    }
    system("pause");
    return 0;
}

打印结果：

在c语言中，指针变量指向的是内存地址，同样数组的内存地址也可以指向，数组名就是数组的首地址。

int len = sizeof(a)/sizeof(int);

前边说了sizeof获取的是内存大小，sizeof(a)获取的是数组a所占的内存大小，sizeof(int)获取的是一个int类型的数据所占的内存大小，数组的内存大小/单个int的内存大小=数组的长度。

指针数组和数组指针

指针数组

int main() {
    char* name[] = {"张三","李四","王五","赵六"};
    int i, n = 4;
    for (i = 0; i < n; i++){
        printf("%s \n", name[i]);
    }
    system("pause");
    return 0;
}

打印结果：

char※ name[]这是一个指针数组，一个叫name的数组，他的返回类型是char※，一个指针。同样，返回类型是指针，数组存的同样也是指针，而指针的含义就是指针变量所指向的内存地址的值。所以可以理解指针数组存的其实就是真实的数据。所以在打印的时候直接打印name[i]，就可以打印出数组的内容。

数组指针

int main() {
    int name1[4] = { 666, 777, 555, 444 };
    int (*name2)[4] = NULL;
    name2 = &name1;
    int i, n = 4;
    for (i = 0; i < 4; i++){
        printf("%d \n", name2[i]);
    }
    system("pause");
    return 0;
}

打印结果：

数组指针顾名思义就是一个数组的指针，数组指针不能直接声明数组元素，前边我们说了，指针指向的是内存地址，所以没法直接赋值。需要用&来获取要指向的内存地址。从打印结果我们可以看出，输出的其实是数组元素的内存地址。那怎么获取值呢？我们套用一下之前的

int a = 10;
int *p;
p = &a;

上边的代码，我们在打印p的值得时候是

printf("%d \n",*p);

这样就可以打印出p所指向的内存地址上的值。

同理，对于
int name1[4] = { 666, 777, 555, 444 };
int (*name2)[4] = NULL;
name2 = &name1;

我们想要打印数组的元素内容的话，就要这么写

printf("%d \n", (*name2)[i]);

整体修改后的代码为：

int main() {
    int name1[4] = { 666, 777, 555, 444 };
    int (*name2)[4] = NULL;
    name2 = &name1;
    int i, n = 4;
    for (i = 0; i < 4; i++){
        printf("%d \n", (*name2)[i]);
    }
    system("pause");
    return 0;
}

函数指针

函数指针看网上说的挺复杂的，因为主要目的是为了ndk开发，就按照最简单的例子来搞吧，到时候遇到问题再细究。

int plus(int a, int b){
    return a + b;
}    

int main(){
    int c = 3;
    int d = 5;
    int result;

    //定义一个函数指针calc
    int(*calc)(int a ,int b);

    //calc函数指针指向plus函数。
    calc = plus;

    //调用函数指针，等同于调用该指针指向的内存地址的函数。
    result = calc(3,5);

    printf("%d \n", result);
    system("pause");
    return 0;
}

打印结果：

函数指针的使用跟我们数组指针的使用一样，套用我们的老套路

int a = 10;
int *p;
p = &a;

只不过这里不用指向函数的内存地址，因为函数名就是函数的首地址，这里跟数组名是数组的首地址一个道理。

calc = plus;

下边我们刚换传入参数，将参数改成指针，再做一下测试，

int plus2(int *a, int *b){
    return *a + *b;
}

int main(){
    int c = 3;
    int d = 5;
    int result;

    //定义一个函数指针calc2
    int(*calc2)(void *a, void *);

    //calc2函数指针指向plus2函数
    calc2 = (int(*)(void *,void *))plus2;

    //调用函数指针，等同于调用该指针指向的内存地址的函数。
    result = calc2(&c, &d);

    printf("%d \n", result);
    system("pause");
    return 0;
}

先来看一下函数指针的参数

int(*calc2)(void *a, void *);

两个没有返回类型的指针，这么写的作用是为了让传入参数方便扩展，比如上边定义的

int plus2(int *a, int *b)

plus2的传入参数就是两个int指针，因为函数指针参数本身没有数据类型，所以我们这边可以随便定义，到时候直接强制转换就行了，很方便。

if (a == NULL){
printf(“内存不够”);
return;
}

    printf("%d \n",sizeof(a));

冒泡排序

以前在学校的时候学javase的时候，老师说java学的好不好，会不会冒泡排序一下子就看出来了，虽然现在发现并没有什么卵用。本着会冒泡排序就标志着学会一门语言的一贯政策，我还是决定去写一下。其实你会发现，如果上边记录的全部都懂了，这玩意很简单。

void sort(int* name[], int length){
    int i;
    int j;
    int  temp;
    for (i = 0; i < length;i++){
        for (j = 0; j < length - i - 1; j++){
            printf("值name[j]:%d \n",name[j]);
            printf("值name[j+1]:%d \n",name[j+1]);
            printf("地址name[j]:%#x \n", name[j]);
            printf("地址name[j+1]:%#x \n", name[j + 1]);
            if (*name[j] > *name[j + 1]){
                temp = *name[j];
                *name[j] = *name[j + 1];
                *name[j + 1] = temp;
            }
        }
    }
}
int main(){
    int m = 200;
    int n = 400;
    int j = 500;
    int x = 100;
    int e = 900;

    int i;
    int *name[] = { &m, &n, &j, &x, &e };
    //int name[5] = { 900, 400, 500, 100, 600 };
    int len = (sizeof(name) / sizeof(int));
    sort(name, len);    
    printf("数组长度：%d \n", len);
    for (i = 0; i < len; i++){
        printf("%d \n",*name[i]);
    }
    system("pause");
    return 0;
}
