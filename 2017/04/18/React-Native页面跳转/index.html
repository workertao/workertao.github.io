React-Native页面跳转

发表于 2017-04-18 | 分类于 react-native |

文字不够，图片来凑……

胡编乱造

android之间的页面跳转和传参用的是intent，页面存在任务堆栈中，最先添加的在最底层，最后添加的在最上层。好在这个思想不变。react-native的页面跳转也需要这样的一个intent。在react-native中文网中有介绍这样一个组件Navigator(即上图)。给我的感觉有点类似于android中的任务堆栈,react-native把android中的任务堆栈给实例成了一个组件(即Navigator)。react-native中的页面添加和删除，也都是直接调用它来操作的。

Navigator

常用属性介绍：

  1. initialRoute：声明需要跳转的页面，可进行传参，传参格式以键值对的形式传递。
  2. renderScene：renderScene需要指定一个函数来渲染页面，函数会被传入route和navigator对象，返回一个需要跳转到的页面。renderScene类似于startActivity，会传入intent，这里传入的route。我们需要将需要将传递的参数和navigator以属性的形式传递到下一个页面。
  3. configureScene：设置页面跳转的动画，官方有提供以下几个。
  • Navigator.SceneConfigs.PushFromRight (默认)
  • Navigator.SceneConfigs.FloatFromRight
  • Navigator.SceneConfigs.FloatFromLeft
  • Navigator.SceneConfigs.FloatFromBottom
  • Navigator.SceneConfigs.FloatFromBottomAndroid
  • Navigator.SceneConfigs.FadeAndroid
  • Navigator.SceneConfigs.HorizontalSwipeJump
  • Navigator.SceneConfigs.HorizontalSwipeJumpFromRight
  • Navigator.SceneConfigs.VerticalUpSwipeJump
  • Navigator.SceneConfigs.VerticalDownSwipeJump

项目实例

setup.js

import React from 'react'
import {
StyleSheet,
Text,
Image,
View,
Navigator,
} from 'react-native'
import HomePage from './HomePage'
export default function setup() {
    class Root extends React.Component {
        renderScene = (route, navigator) => {
        //获取要跳转的页面
        let Traget = route.component;
        //{...route.params}是将当前对象的所有属性都传递到下个页面。
        // navigator传递到下个页面为了下个页面能够进行跳转或者回退
        return <Traget {...route.params} navigator={navigator}/>
    }

    render() {
        //initialRoute:声明要跳转的页面  params:要传递的参数，参数以键值对的形式传递
        //renderScene:返回要跳转的页面
        //configureScene：设置页面跳转的动画
        return <Navigator
            initialRoute={{component:HomePage,params:{title:'张三的歌'}}}
            renderScene={(route,navigator)=>this.renderScene(route,navigator)}
            configureScene={route=>Navigator.SceneConfigs.FadeAndroid}
        >
        </Navigator>
        }
    }
return <Root/>
}

HomePage.js

import React from 'react'
import {
StyleSheet,
Text,
Image,
View,
} from 'react-native'
import MyPage from './MyPage'
export default class HomePage extends React.Component {
    goToMyPage = () => {
        //添加到任务堆栈中MyPage页面，传递参数title
        this.props.navigator.push({
        component: MyPage,
        params: {title: '我的页面'}
        })
    }

    render() {
        return <View style={styles.container}>
            <View style={styles.titleStyle}>
                /* 调用当前对象的属性props来接收传递的参数title*/
                <Text style={styles.fontStyle}>{this.props.title}</Text>
            </View>
            <View style={styles.centerStyle}>
                <Text style={styles.fontStyle} onPress={this.goToMyPage}>{`let's go MyPage`}</Text>
            </View>
            </View>
        }
    }
    const styles = StyleSheet.create({
        container: {
        flex: 1,
            flexDirection: 'column',
            backgroundColor: '#FFF',
        },
        titleStyle: {
            flexDirection: 'row',
            alignSelf: 'center',
        },
        centerStyle: {
            flex: 1,
            flexDirection: 'column',
            justifyContent: 'center',
            alignItems: 'center',
        },
        fontStyle: {
            color: '#F00',
            fontSize: 24,
        }
    })

MyPage.js

`import React from 'react'
import {
StyleSheet,
Text,
Image,
View,
} from 'react-native'
export default class MyPage extends React.Component {
doBack = () => {
    //将当前页面从任务堆栈中移除
    this.props.navigator.pop();
}

render() {
    return <View style={styles.container}>
        <View style={styles.titleStyle}>
            /* 调用当前对象的属性props来接收传递的参数title*/
            <Text style={styles.fontStyle}>{this.props.title}</Text>
        </View>
        <View style={styles.centerStyle}>
            <Text style={styles.fontStyle} onPress={this.doBack}>{`back`}</Text>
        </View>

    </View>
}
}
const styles = StyleSheet.create({
    container: {
        flex: 1,
        backgroundColor: '#FF0',
        flexDirection: 'column',
        justifyContent: 'center',
           alignItems: 'center',
    },
    titleStyle: {
        flexDirection: 'row',
        alignSelf: 'center',
    },
    centerStyle: {
        flex: 1,
        flexDirection: 'column',
        justifyContent: 'center',
        alignItems: 'center',
    },
    fontStyle: {
        color: '#F00',
        fontSize: 24,
    }
})`
