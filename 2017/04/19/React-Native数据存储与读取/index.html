AsyncStorage

AsyncStorage是一个简单的、异步的、持久化的Key-Value存储系统，它对于App来说是全局性的

方法

setItem

tatic getItem(key: string, callback?: ?(error: ?Error, result: ?string) => void)

读取key字段并将结果作为第二个参数传递给callback。如果有任何错误发生，则会传递一个Error对象作为第一个参数。返回一个Promise对象。

getItem

static setItem(key: string, value: string, callback?: ?(error: ?Error) => void)

将key字段的值设置成value，并在完成后调用callback函数。如果有任何错误发生，则会传递一个Error对象作为第一个参数。返回一个Promise对象。

removeItem

static removeItem(key: string, callback?: ?(error: ?Error) => void)

删除一个字段。返回一个Promise对象。

mergeItem

static mergeItem(key: string, value: string, callback?: ?(error: ?Error) => void)

假设已有的值和新的值都是字符串化的JSON，则将两个值合并。返回一个Promise对象。还没有被所有原生实现都支持。

clear

static clear(callback?: ?(error: ?Error) => void)

删除全部的AsyncStorage数据，不论来自什么库或调用者。通常不应该调用这个函数——使用removeItem或者multiRemove来清除你自己的key。返回一个Promise对象。

代码示例

import React, {Component} from 'react';
import {
    StyleSheet,
    Text,
    View,
    Navigator,
    AsyncStorage,
    Button,
} from 'react-native';
export default class Test extends React.Component {
    constructor(props) {
        super(props);
        this.state = {
            data: [
                {
                    name: '张三',
                    age: 18
                },
                {
                    name: '李四',
                    age: 19
                },
                {
                    name: '王五',
                    age: 20
                },
                {
                    name: '赵六',
                    age: 21
                },
            ],
            saveText: '还未保存数据',
            readeText: '还未读取数据',
            clearText: '还未清空数据'
        }
    }

    /**
     * 写入
     */
    onSave = () => {
        AsyncStorage.setItem("data", JSON.stringify(this.state.data))
            .then(value => {
                console.log("保存成功：" + value);
                this.setState({saveText: "保存成功"});
            })
    }
    /**
     * 读取
     */
    onRead = () => {
        AsyncStorage.getItem("data")
            .then(value => {
                console.log("读取数据：" + value);
                this.setState({readeText: value === null ? '请添加数据' : value});
            })
    }

    /**
     * 清空
     * @returns {XML}
     */
    onRemoveItem = () => {
        AsyncStorage.removeItem("data")
            .then(value => {
                console.log("清空成功：" + value);
                this.setState({clearText: '清空成功'});
            })
    }
    /**
     * 清空
     * @returns {XML}
     */
    onClear = () => {
        AsyncStorage.clear()
            .then(() => {
                console.log("清空成功：" + value);
                this.setState({clearText: '清空成功'});
            })
    }

    render() {
        return <View style={styles.container}>
            <Button onPress={this.onSave} style={styles.button} title='保存'></Button>
            <Text style={styles.saveText}>{this.state.saveText}</Text>
            <Button onPress={this.onRead} style={styles.button} title='读取'></Button>
            <Text style={styles.readText}>{this.state.readeText}</Text>
            <Button onPress={this.onRemoveItem} style={styles.button} title='清空'></Button>
            <Text style={styles.readText}>{this.state.clearText}</Text>
        </View>
    }
}
const styles = StyleSheet.create({
    container: {flex: 1, flexDirection: 'column'},
    button: {width: 100, height: 80},
    saveText: {fontSize: 18, color: '#00F'},
    readText: {fontSize: 18, color: '#F00'}
})
